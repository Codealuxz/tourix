# Tourix

Tower defense game built entirely in **vanilla HTML, CSS and JavaScript**. No framework, no build step, no dependency.

**Play it here: [tower.codealuxz.fr](https://tower.codealuxz.fr/)**

## How to play

- Place towers along the path to stop the waves of enemies
- Each kill earns money to buy and upgrade more towers
- Survive as many waves as you can

## Run locally

No build needed, it is a static site:

```bash
git clone https://github.com/Codealuxz/tourix.git
cd tourix
```

Then open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Stack

- HTML / CSS / JavaScript, 100% vanilla
- Canvas rendering for the game loop
